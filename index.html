Typography in CSS

Name Description
font-family The font of an element can be changed using this property. When specifying a typeface with multiple words, like Times New Roman, it is recommended to use quotation marks (' ') to group the words together.
Web Safe Fonts There is a selection of fonts that will appear the same across all browsers and operating systems.
Fallback Fonts and Font Stacks Web safe fonts are good fallback fonts that can be used if your preferred font is not available.
Serif and Sans-Serif You may be wondering what features make a font similar to another font. The fonts Caslon, Georgia, and Times New Roman are Serif fonts. Serif fonts have extra details on the ends of each letter, as opposed to Sans-Serif fonts, which do not have the extra details.
font-weight The font-weight property can take any one of these keyword values: bold, normal, lighter, bolder. Numerical values can range from 1 (lightest) to 1000 (boldest), but it is common practice to use increments of 100.
text-transform Text can also be styled to appear in either all uppercase or lowercase.
font-style You can also italicize text with the font-family property.
